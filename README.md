# Sistema de Gestión Hospitalaria

## Descripción
Este sistema permite gestionar pacientes, médicos y citas médicas mediante un enfoque orientado a objetos.

El sistema implementa operaciones CRUD (Crear, Leer, Actualizar y Eliminar) para las entidades principales, 
y utiliza archivos CSV para la persistencia de datos.

## Objetivo
Facilitar la administración de información hospitalaria mediante estructuras de programación limpias y reutilizables.

## Funcionalidades
- Registro de pacientes
- Consulta de pacientes
- Actualización de datos
- Eliminación de registros
- Persistencia en archivos CSV

## Tecnologías utilizadas
- C#
- Programación Orientada a Objetos
- CSV
